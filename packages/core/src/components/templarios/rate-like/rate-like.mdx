## Eventos

### tpChange

- O evento **tpChange** é emitido quando o usuário clica em um dos botões.

- Se o usuário clica no botão de like, o evento emitido possui a propriedade `detail="like"`.

- Se o usuário clica no botão de dislike, o evento emitido possui a propriedade `detail="dislike"`.
